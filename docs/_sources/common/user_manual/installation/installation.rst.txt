Installation
================

To get the code:

.. code-block:: none 

    git clone https://github.com/NHERI-SimCenter/BRAILS.git

    cd BRAILS 
 
    pip install -r requirements.txt 




The above pip command will install:

::

    requests
    numpy
    geopandas
    scipy
    matplotlib
    tensorflow
    seaborn
